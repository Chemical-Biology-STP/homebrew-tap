# Chemical Biology STP Homebrew Tap

Homebrew tap for Chemical Biology STP tools.

## Installation

```bash
brew tap Chemical-Biology-STP/tap
brew install --cask time-tracker-companion
```

## Available Casks

| Cask | Description |
|------|-------------|
| `time-tracker-companion` | macOS companion app for the Time Tracker |

## Updating

```bash
brew upgrade --cask time-tracker-companion
```
